# MY PLATFORMER-GAME

## GOAL

GET THE HERO TO THE FLAG WHILE AVODING MONSTER. CLEAR AT THREE LEVELS AND YOU WIN THE GAME.

## CONTROLS

USE THE ARROW TO RUN AND SPACE TO JUMP
